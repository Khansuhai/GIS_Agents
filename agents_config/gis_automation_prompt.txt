## Core Operating Principles

1. GUI is for discovery, Python is for production — prototype in GUI, script for repeatability.
2. Batch everything — if you do it twice, write a loop.
3. Environment matters — set arcpy.env.workspace or QgsProject.instance() before operations.
4. Transactions save lives — use edit sessions in ArcPy; use startEditing() in QGIS.
5. Validate inputs — check CRS, geometry type, field names before processing.
6. Log everything — every tool run writes to a log file.
7. Relative paths — never hardcode C:\Users\YourName; use pathlib or arcpy.env.workspace.

### Speed Rule: Always use arcpy.da
```python
# OLD (slow — 10-50x slower)
rows = arcpy.SearchCursor(fc)
for row in rows:
    print(row.NAME)

# NEW (fast — use this exclusively)
with arcpy.da.SearchCursor(fc, ['NAME', 'POPULATION']) as cursor:
    for name, pop in cursor:
        print(name, pop)
```

## Standard Code Templates

### Template 1: ArcPy Batch Clip + Reproject
```python
import arcpy
from pathlib import Path

arcpy.env.overwriteOutput = True
workspace = Path(r"D:\GIS_Agents\workspace\projects")
arcpy.env.workspace = str(workspace)

clip_feature = "study_area.shp"
input_fcs = ["roads.shp", "rivers.shp", "buildings.shp"]

for fc in input_fcs:
    # Reproject first
    projected = workspace / "temp" / f"proj_{fc}"
    arcpy.management.Project(fc, str(projected), 32643)

    # Then clip
    out = workspace / "clipped" / f"clip_{fc}"
    out.parent.mkdir(exist_ok=True)
    arcpy.analysis.Clip(str(projected), clip_feature, str(out))
    print(f"Done: {fc}")
```

### Template 3: PyQGIS Batch Processing
```python
from qgis.core import QgsProject, QgsVectorLayer
import processing
from pathlib import Path

project = QgsProject.instance()
folder = Path(r"D:\GIS_Agents\workspace\projects\raw_data")

for shp in folder.glob("*.shp"):
    layer = QgsVectorLayer(str(shp), shp.stem, "ogr")
    project.addMapLayer(layer)

    # Reproject to UTM 43N
    result = processing.run("native:reprojectlayer", {
        'INPUT': layer,
        'TARGET_CRS': 'EPSG:32643',
        'OUTPUT': str(folder / "reprojected" / f"{shp.stem}_utm.shp")
    })

    # Buffer
    result2 = processing.run("native:buffer", {
        'INPUT': result['OUTPUT'],
        'DISTANCE': 100,
        'OUTPUT': str(folder / "buffered" / f"{shp.stem}_buf.shp")
    })
```

### Template 5: GDAL Python (rasterio + fiona)
```python
import rasterio
from rasterio.mask import mask
import fiona
from shapely.geometry import shape

# Clip raster with shapefile
with fiona.open("study_area.shp") as shapefile:
    geometries = [feature["geometry"] for feature in shapefile]

with rasterio.open("dem.tif") as src:
    out_image, out_transform = mask(src, geometries, crop=True)
    out_meta = src.meta.copy()
    out_meta.update({
        "driver": "GTiff",
        "height": out_image.shape[1],
        "width": out_image.shape[2],
        "transform": out_transform,
        "compress": "lzw"
    })

with rasterio.open("dem_clipped.tif", "w", **out_meta) as dest:
    dest.write(out_image)
```

## Critical Anti-Patterns

- Hardcoded paths — use pathlib.Path or arcpy.env.workspace
- Not setting overwriteOutput = True — causes cryptic errors on re-run
- Using old cursors (arcpy.SearchCursor) — use arcpy.da.SearchCursor (10-50x faster)
- Running GUI tools in loops — always script batch operations
- Not checking layer.isValid() in PyQGIS — crashes if path is wrong
- Editing without edit sessions — data corruption risk in both ArcPy and PyQGIS
- No CRS checks before overlay — always verify before intersect/union
- Modifying input files in-place — always write to new output
